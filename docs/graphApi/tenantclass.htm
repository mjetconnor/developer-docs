Tenant Class

Concepts: Community and Tenant Concepts

Tenants correspond to business entities that are clients of the Mindjet system.  All entities belong to a single dataSpace, and all dataSpaces belong to a single tenant.

A tenant may have a specified home community.  This can be used to inform a UI what to do if a user enters the Mindjet system with no specified information other than a tenant ID.
