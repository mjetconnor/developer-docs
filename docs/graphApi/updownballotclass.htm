UpDownBallot Class

Concepts: Assessment Concepts

A ballot that may have “up”, “down” or “skip” votes, represented by numeric valuations of “1”, “-1” and “0” respectively.

There are no defined properties for instances of this type other than the "type" property itself.  That is enough to identify what kind of assessment value is expected.

Subclass of: Ballot
