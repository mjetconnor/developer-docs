Tutorial

This tutorial is an introduction for how to use the SpigitGraph API to access your data and execute some basic posts.

You should be familiar with the HTTP protocol and its GET, PUT and POST methods.  A familiarity with the JSON interchange format.  And of course you should have at least a basic familiarity with your Spigit application.
