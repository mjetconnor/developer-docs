Classes by Category

This topic provides an index that groups classes broadly according to a category of usage.

 

Categories:

Primitive The primitive classes
General Classes that  have a general usage that could apply broadly in many situations.
Infrastructure Classes that are involved with the basic structure and orientation of the graph.
Agent Classes that relate to various aspects of actors in the system.  These classes draw heavily on the FOAF (Friend Of A Friend) standard vocabulary.
Idea Classes that relate to the representation and management of ideas.
Collaboration

Classes that describe collaborations of various sorts.
Assessment Classes that are used to perform and track assessments.
