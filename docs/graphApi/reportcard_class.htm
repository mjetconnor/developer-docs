ReportCard Class

Concepts: Idea Concepts

Version 3.8.0 and later

ReportCards provide information about how an idea is progressing in relation to graduation criteria, as well as a graduation history.

 

Property

Datatype

Req

Description

activity

activities

  PlainObject

[ * ]

Each element of this collection identifies a graduation event for the idea.  The elements are PlainObjects that have these properties:

stage Stage The stage to which the idea graduated.
date

DateTime

 When the graduation occurred.

 

completed

 PlainObject

SHOULD

For every graduation criterion in effect, there is a property in this object whose name is the graduation criterion name and whose value is the percent completion as a decimal number from 0.0 to 1.0.

In addition, there is a property named "overall" whose value is the percent completion of the idea as a whole, with all individual criteria factored together.

See the Idea class for the list of defined graduation criterion properties.

governor

 Governor

RULE

The governor for the idea described by this reportCard.
