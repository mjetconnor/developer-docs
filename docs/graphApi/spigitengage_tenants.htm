SpigitGraph API   

Each Spigit Customer is established as at least one tenant in our cloud infrastructure.   A tenant is a dedicated set of services and a data repository that supply the infrastructure for Spigit in support of a specific customer.    The tenant is addressable via a Spigit sub-domain or in some cases via a vanity URL.  For example:

 

Spigit Sub-domain http://www.mycompany.spigit.com
Vanity URL

http://innovation.mycompany.com

 

In the case of vanity URLs, they are redirected to a spigit.com sub-domain.
