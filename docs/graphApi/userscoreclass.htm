UserScore Class

A facet used for tracking status information about an Agent.

Subclass of: Facet

 

Property

Datatype

Req

Description

base

Agent

MUST

The Facet class defines the base property.  For UserScores, this MUST be an Agent  This is not strictly limited to User entities, but users are the most common type of agent that has scores.

context

Thing

MUST

The Facet class defines the context property.  

reputation

Decimal

MAY

A number that represents the reputation of the user within this context.  A higher value indicates that the user is held in higher esteem.

points

Decimal

MAY

The user's game points balance within this context.

idea

ideas

Idea

[ * ]

The ideas created by this user in this context.

ideaCount

Integer

RULE

The number of ideas created by this user in this context.

postCount

Integer

RULE

The number of posts executed by this user in this context.

excludeFromLeaderboard

Boolean

MAY

 If specified and has the value "true", this user and its scores should not be included in leaderboards.  

viewUrl

URI

MAY

The URL of the page in the Spigit webapp that displays the profile page for this user in the context of this score's inquiry.  Spigit user profile pages are all in the context of a Spigit community or challenge, User entities therefore do not have a "viewUrl".
