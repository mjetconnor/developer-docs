SpigitGraph and Spigit

This help doc, taken as a whole, describes a landscape of data that is not limited to Spigit. Except for parts of the tutorial, the data model concepts and class definitions are not restricted to what Spigit does today, but also apply to future versions of Spigit as well as future enterprise social innovation applications yet to be conceived and developed.  But for now, Spigit is the primary source of SpigitGraph data.

This section is the only section that discusses and defines the elements of the SpigitGraph API as they are specifically served by Spigit data sources.

Spigit is an enterprise innovation platform designed to encompass the full breadth and scope of an enterprise scale innovation program.  The application is cloud based, leverages social networking and gaming technologies to engage the crowd in innovation.  Spigit provides a sophisticated model for graduating ideas based on popularity, engagement, expert review, and evaluations.
