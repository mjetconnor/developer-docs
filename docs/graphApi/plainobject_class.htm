PlainObject Class

A generic object value.  When elements of the SpigitGraph data model are defined to have this datatype, there is no specific expectation about what properties the object will or may have.


If you are a Javascript programmer, think of PlainObjects as being associative arrays, in the mold of all plain Javascript Objects.  If you are a Java programmer, think of PlainObjects as corresponding to the class Map<String,Object>, where the key is a String (the property  name), and the value may be of any type.
